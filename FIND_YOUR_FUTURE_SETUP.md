# คู่มือระบบ Find Your Future — สภานักเรียน RB

หน้า "Find Your Future" รวมค่าย ทุน กิจกรรม โอกาสเรียนต่อ
ดึงข้อมูลจาก Google Sheet โดยตรง — สภาแก้ Sheet → เว็บอัปเดตเองในไม่กี่นาที

---

## ขั้นที่ 1 — สร้าง Google Sheet

ไปที่ https://sheets.google.com สร้างไฟล์ใหม่ ตั้งชื่อ เช่น "RB Find Your Future"

ใส่หัวคอลัมน์แถวแรก (ตัวเล็กทั้งหมด สะกดตรงเป๊ะ):

```
title | category | detail | link | date | date_iso
```

> ลำดับสลับได้ ระบบอ่านตามชื่อหัว

### ความหมาย:

| คอลัมน์ | บังคับ | ตัวอย่าง | อธิบาย |
|--------|--------|----------|--------|
| `title` | ✅ | "ค่ายโอลิมปิกวิชาการ" | ชื่อกิจกรรม |
| `category` | ❌ | `วิชาการ`, `กิจกรรม`, `อาสา`, `อื่นๆ` | หมวดสำหรับปุ่มกรอง |
| `detail` | ❌ | "ค่ายอบรมเข้มข้น 5 วัน..." | รายละเอียดเพิ่มเติม |
| `link` | ❌ | `https://...` | ลิงก์ไปเว็บสมัคร/รายละเอียด |
| `date` | ❌ | "26 พ.ค. 2569" หรือ "28 พ.ค. – 14 มิ.ย. 2569" | วันที่แบบไทย (ใช้แสดง + คำนวณ pill) |
| `date_iso` | ❌ | `2026-05-26` หรือ `2026-05-28~2026-06-14` | วันที่แบบ ค.ศ. (ใช้แทน date ถ้าอยากแม่นยำ) |

### หมวดที่รองรับ (ปุ่มกรอง):

- **ทั้งหมด** (auto)
- **วิชาการ** — ค่ายเรียน, ตวลิม, ทุน
- **กิจกรรม** — workshop, อบรม
- **อาสา** — จิตอาสา, ค่ายอาสาพัฒนา
- **อื่นๆ** — อะไรก็ได้ที่ไม่เข้าหมวดอื่น

> ถ้าใส่ category ชื่ออื่นใน Sheet ก็จะมีปุ่มกรองขึ้นอัตโนมัติ

### ✨ ระบบ pill อัตโนมัติ:

ระบบคำนวณ pill จาก `date` ให้เอง:

| สถานการณ์ | pill |
|-----------|------|
| อีก 5 วัน | 🟡 "อีก 5 วัน" |
| พรุ่งนี้ | 🟡 "พรุ่งนี้" |
| ช่วงวัน, กำลังเปิด | 🟡 "กำลังเปิด · เหลือ X วัน" |
| ผ่านไปแล้ว | ⚫ "ผ่านมา X วัน" |
| เหลือ >30 วัน | 🌸 "อีก X วัน" |
| ไม่มีวันที่ | 🌸 "ติดตามประกาศ" |

## ขั้นที่ 2 — publish เป็น CSV

1. File → Share → Publish to web
2. ช่องซ้าย: เลือกชีต
3. ช่องขวา: Comma-separated values (.csv)
4. กด Publish → copy ลิงก์

---

## ขั้นที่ 3 — ใส่ลิงก์ในเว็บ

1. เปิดไฟล์ `find-your-future.html`
2. หาบรรทัด:
   ```js
   const SHEET_CSV_URL = '';
   ```
3. วางลิงก์ CSV
4. push ขึ้น GitHub ครั้งเดียว

หลังจากนี้แก้ใน Sheet ได้ตลอด ไม่ต้อง push อีก

---

## ใช้งานหลังจากนี้

- เพิ่มกิจกรรมใหม่: เพิ่มแถวใน Sheet → เว็บอัปเดตเอง 1-5 นาที
- ลบ/แก้: แก้ใน Sheet ตรงๆ
- กิจกรรมผ่านไปแล้ว: ระบบจัดให้อยู่ล่างสุดสีเทาเอง
